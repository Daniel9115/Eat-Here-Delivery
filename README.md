# EAT HERE | Delivery

Um site de delivery de alimentos com uma interface simples e acessível, permitindo que os usuários façam pedidos, busquem produtos e obtenham informações sobre a empresa.

## 🖥️ Visualização

[Clique aqui](https://daniel9115.github.io/Eat-Here-Delivery/index.html) para visualizar o site.

## 🚀 Funcionalidades

- **Menu de navegação**: Navegação simples com links para páginas de busca, pedidos, contato, cadastro e quem somos.
- **Busca de produtos**: Interface para pesquisa de itens no menu.
- **Informações de contato**: Exibição do horário de funcionamento e número de telefone para contato.
- **Responsividade**: O site foi desenvolvido para ser visualizado em dispositivos de desktop.
- **Rodapé**: Contém informações de direitos autorais e créditos.

## 🛠️ Tecnologias Usadas

- **HTML5**: Estruturação do conteúdo da página.
- **CSS3**: Estilos e design da página.
- **Google Fonts**: Uso das fontes **Roboto** e **Allerta Stencil**.
- **Flaticon**: Ícones usados no site (como a lupa e a imagem do talher).
